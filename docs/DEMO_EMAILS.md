## Brevo Automation Setup Instructions

### Step 1: Create Contact List
1. Go to Brevo Dashboard → Contacts → Lists
2. Create new list: "Team Mehrwert - Landing Page Signups"
3. Note the List ID (use in config.php)

### Step 2: Create Email Templates
1. Go to Campaigns → Email Templates
2. Create 5 templates using the content above
3. Use these variables:
   - `{{FIRSTNAME}}` - First name
   - `{{LASTNAME}}` - Last name
   - `{{EMAIL}}` - Email address
   - `{{MENTOR}}` - Mentor name
   - `{{PHONE}}` - Phone number

### Step 3: Set Up Automation Workflow
1. Go to Automation → Create a new workflow
2. **Trigger:** Contact added to list "Team Mehrwert - Landing Page Signups"
3. **Actions:**
   - Send Email 1 immediately
   - Wait 2 days → Send Email 2
   - Wait 3 days → Send Email 3
   - Wait 3 days → Send Email 4
   - Wait 6 days → Send Email 5

### Step 4: Configure Contact Attributes
Add these custom attributes in Brevo:
- `FIRSTNAME` (text)
- `LASTNAME` (text)
- `MENTOR` (text)
- `PHONE` (text)
- `SIGNUP_DATE` (date)

### Step 5: Test the Workflow
1. Add a test contact to the list
2. Verify all emails are sent at correct intervals
3. Check personalization variables work correctly
